Ringkasan Project

Project ini melatih model CNN pada dataset gambar hewan dengan 8 kelas, yaitu: cane, cavallo, elefante, farfalla, gallina, gatto, mucca.

Model dilatih dengan arsitektur CNN yang terdiri dari beberapa blok konvolusi, pooling, batch normalization, dan dropout.

File Penting

  • train_cnn.py - skrip pelatihan CNN
  • cnn_model.h5 - model terlatih
  • predict.py - skrip prediksi gambar
  • cnn-animals.ipynb - notebook analisis dan visualisasi
  • report.html - laporan lengkap dengan hasil
  • project-report.zip - download paket laporan dan kode

Cara Menjalankan

  1. Siapkan environment Python dan pasang dependensi:
    2. pip install -r requirements.txt
  2. Latih model dengan:
    3. python train_cnn.py
  3. Jalankan prediksi gambar dengan:
    4. python predict.py dataset/gatto/1.jpeg

Catatan

Pastikan folder dataset/ berisi subfolder untuk setiap kelas. Model menggunakan ukuran input 128×128 piksel.
